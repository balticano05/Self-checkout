# Self-checkout
Develop a desktop self-checkout application using C# and WPF. The app will streamline purchasing by allowing customers to manage their own checkout, similar to retail self-checkout systems. Features include adding products to a cart, applying discounts, and finalizing purchases. Use CSV files for the database.

# Dependencies and tools:

    - .net core 8.0
    - WPF
    - iTextSharp 5.5.13.4
    - System.Text.Encoding.CodePages 9.0.0
    - csv
    - PDF
